# Weave CLI

> CLI for use with Weave.


## Usage

```
Usage:
  $ weave <path/to/template> <options>

Options:
  --pages       Parse the JSON template to pages
  --partials    Parse the JSON template to partials

Example:
  $ weave test/tester-partials.json --partials
```
